# Student Management System

## Aim
To develop a Student Management System using C++ and File Handling.

## Description
This project allows users to manage student records efficiently using file handling concepts in C++.

## Features
- Add Student Record
- Search Student Record
- Display Student Record
- Update Student Record
- Delete Student Record

## Technologies Used
- C++
- File Handling
